# 📈 주요국 통화의 대원화 환율 변동 추이 및 통계적 분석 대시보드

본 프로젝트는 미국 달러(USD), 일본 엔(JPY), 중국 위안(CNY)의 대원화 환율 데이터를 기반으로 시계열 추세와 통계적 리스크를 심층 분석하는 **Streamlit 기반 멀티 페이지 웹 애플리케이션**입니다. 

경제 및 통계학 탐구 보고서 형태로 구성되어 있으며, 복잡한 인코딩 설정이나 데이터 전처리 없이 웹 브라우저에서 직관적인 시각화 지표를 제공합니다.

---

## 🚀 주요 기능 및 페이지 구성

사이드바 내비게이션을 통해 총 4개의 독립된 분석 페이지를 제공합니다.

1. **🏠 홈 (종합 분석)**
   * **기초 통계량:** 선택 기간 내 각 통화의 관측치, 평균값, 변동성(표준편차), 최솟값, 최댓값 요약
   * **상관관계 검증:** 피어슨 상관계수($Pearson\ Correlation$)를 활용한 통화 간 연동성 분석
   * **리스크 비교:** 최근 150거래일간의 일일 변동률 시계열 비교 및 최고 위험 통화 자동 판정
2. **🇺🇸 미국 (원/달러)**
   * **이동평균선(MA) 분석:** 실제 환율과 20일·60일 이평선 시각화를 통한 단기/장기 트렌드 분석
   * **변동성 분포:** 일일 변동률 히스토그램($Histogram$)을 통해 리스크 밀집도 파악
3. **🇯🇵 일본 (원/100엔)**
   * 엔화 환율의 이평선 교차 지점 및 전용 변동성 분포 차트 제공
4. **🇨🇳 중국 (원/위안)**
   * 위안화 환율의 이평선 추세 및 통계적 위험도 지표 제공

---

## 🛠️ 핵심 해결 기술 (Tech Stack)

* **Streamlit Single-File Multi-Page:** `pages/` 폴더 분리 없이 `st.sidebar.radio`를 활용한 쾌적한 화면 전환 시스템 구현
* **지능형 인코딩 감지:** 한국은행 ECOS 및 Excel에서 파생되는 한글 CSV 인코딩 문제(`UnicodeDecodeError`)를 해결하기 위해 `UTF-8`, `CP949`, `EUC-KR` 자동 교차 검증 로직 내장
* **시각적 노이즈 제거:** 데이터 단위를 고려한 독립 스케일링 그래프 및 눈이 편안한 표준 3색 챠트 테마 적용
* **경로 최적화:** 사용자가 수동으로 업로드하지 않아도 서버 내의 데이터를 자동으로 탐색하여 즉시 구동
