# 📈 주요국 통화의 대원화 환율 변동 추이 및 통계적 분석 대시보드

본 프로젝트는 미국 달러(USD), 일본 엔(JPY), 중국 위안(CNY)의 대원화 환율 데이터를 기반으로 시계열 추세와 통계적 리스크를 심층 분석하고, 미래 장기 추세를 예측하는 **Streamlit 기반 멀티 페이지 웹 애플리케이션**입니다. 

"환율은 점점 오르고 있을까?"라는 핵심 질문을 바탕으로 고급 금융공학 기법을 활용한 2030년 장기 시뮬레이션 모델을 제공하여 경제/금융/통계학 탐구 보고서 및 시각화 대시보드로 활용하기 좋습니다.

---

## 🚀 주요 기능 및 페이지 구성

사이드바 내비게이션을 통해 총 5개의 독립된 분석 및 탐구 단계를 제공합니다.

1. **🏠 홈 (종합 분석)**
   * **기초 통계량:** 선택 기간 내 각 통화의 관측치, 평균값, 변동성(표준편차), 최솟값, 최댓값 요약
   * **상관관계 검증:** 피어슨 상관계수($Pearson\ Correlation$)를 활용한 통화 간 연동성 분석
   * **리스크 비교:** 최근 150거래일간의 일일 변동률 시계열 비교 및 최고 위험 통화 자동 판정

2. **🔮 2030년 장기 환율 예측 (핵심 기능)**
   * **금융공학 모델 적용:** 장기 예측 시 발생하는 통계적 왜곡을 방지하기 위해 **기하 브라운 운동(Geometric Brownian Motion, GBM)** 모델 적용
   * **몬테카를로 시뮬레이션:** 과거 데이터의 평균 상승률(Drift)과 변동성(Volatility)을 추출하여 무작위 시나리오 연산
   * **부채꼴 리스크 범위 제시:** 중간값 예측선과 함께 시간이 흐를수록 누적되는 미래 불확실성을 반영한 **최대 폭등/폭락 시나리오(경계선)** 시각화

3. **🇺🇸 미국 (원/달러)**
   * 실제 환율과 20일·60일 이동평균선(MA)을 통한 단기/장기 트렌드 분석 및 눈이 편안한 전용 일일 변동률 히스토그램($Histogram$) 제공

4. **🇯🇵 일본 (원/100엔)**
   * 엔화 환율의 이평선 추세 및 가독성을 높인 독립형 변동성 분포 차트 제공

5. **🇨🇳 중국 (원/위안)**
   * 위안화 환율의 추세선 및 리스크 분포 데이터 제공

## 🛠️ 핵심 해결 기술 (Tech Stack)

* **Streamlit Multi-Page Architecture:** 파일 분리 없이 단일 파일 내에서 `st.sidebar.radio` 제어문을 활용한 쾌적한 화면 전환 시스템 구현
* **기하 브라운 운동 모델 ($GBM$):** 외부 머신러닝 라이브러리 의존성 없이 `numpy` 수학 연산만을 활용해 월스트리트 표준의 장기 자산 가격 시뮬레이션 모델 구현
* **소프트 표준 3색 테마:** 시각적 피로도를 낮추는 디프 블루, 소프트 오렌지, 차분한 그린 컬러셋을 적용하여 눈이 편안한 히스토그램 및 차트 레이아웃 제공
* **지능형 인코딩 예외 처리:** 한국은행 ECOS 데이터 특유의 한글 CSV 인코딩 문제(`UnicodeDecodeError`)를 해결하기 위해 다중 인코딩 디코딩 로직 내장
