# 📈 주요국 통화의 대원화 환율 변동 추이 및 통계적 분석 대시보드

본 프로젝트는 미국 달러(USD), 일본 엔(JPY), 중국 위안(CNY)의 대원화 환율 데이터를 기반으로 시계열 추세와 통계적 리스크를 심층 분석하고 미래 추세를 예측하는 **Streamlit 기반 멀티 페이지 웹 애플리케이션**입니다. 

"환율은 점점 오르고 있을까?"라는 핵심 질문을 바탕으로 통계학적 증명 및 단기 예측 모델을 제공하여 경제/통계학 탐구 보고서로 활용하기 좋습니다.

---

## 🚀 주요 기능 및 페이지 구성

사이드바 내비게이션을 통해 총 5개의 독립된 분석 및 탐구 단계를 제공합니다.

1. **🏠 홈 (종합 분석)**
   * **기초 통계량:** 선택 기간 내 각 통화의 관측치, 평균값, 변동성(표준편차), 최솟값, 최댓값 요약
   * **상관관계 검증:** 피어슨 상관계수($Pearson\ Correlation$)를 활용한 통화 간 연동성 분석
   * **리스크 비교:** 최근 150거래일간의 일일 변동률 시계열 비교 및 최고 위험 통화 자동 판정

2. **🔮 미래 환율 예측 탐구 (핵심 기능)**
   * **추세 분석:** 선택한 기간 동안 환율이 통계적으로 상승세인지 하락세인지 정량적으로 진단
   * **선형 회귀 예측:** 과거 데이터의 기울기를 바탕으로 **향후 30거래일 동안의 미래 환율 변동 궤적**을 직선 추세선($Trendline$)으로 시각화 및 수치 예측

3. **🇺🇸 미국 (원/달러)**
   * 실제 환율과 20일·60일 이동평균선(MA)을 통한 단기/장기 트렌드 분석 및 일일 변동률 히스토그램 제공

4. **🇯🇵 일본 (원/100엔)**
   * 엔화 환율의 이평선 추세 및 가독성을 높인 전용 변동성 분포($Histogram$) 차트 제공

5. **🇨🇳 중국 (원/위안)**
   * 위안화 환율의 추세선 및 리스크 분포 데이터 제공

---

## 🛠️ 핵심 해결 기술 (Tech Stack)

* **Streamlit Multi-Page Architecture:** 파일 분리 없이 단일 파일 내에서 `st.sidebar.radio` 제어문을 활용한 쾌적한 화면 전환 시스템 구현
* **선형 회귀 분석 모델 ($Linear\ Regression$):** 외부 머신러닝 라이브러리 의존성 없이 `numpy.polyfit` 모델을 활용해 순수 통계적 데이터를 바탕으로 한 신뢰성 높은 추세 연장선 구현
* **소프트 표준 3색 테마:** 시각적 피로도를 낮추는 디프 블루, 소프트 오렌지, 차분한 그린 컬러셋을 적용하여 눈이 편안한 히스토그램 레이아웃 제공
* **지능형 인코딩 예외 처리:** 한국은행 ECOS 데이터 특유의 한글 CSV 인코딩 문제(`UnicodeDecodeError`)를 해결하기 위해 다중 인코딩 디코딩 로직 내장
