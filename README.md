# 📈 환율의 변화와 예측 대시보드

본 프로젝트는 미국 달러(USD), 일본 엔(JPY), 중국 위안(CNY)의 대원화 환율 데이터를 기반으로 환율의 역사적 변화 흐름을 추적하고, 미래의 변동 범위를 통계적으로 시뮬레이션하는 **Streamlit 기반 멀티 페이지 웹 애플리케이션**입니다.

"환율은 점점 오르고 있을까?"라는 질문을 시작으로, 과거의 변화 패턴 분석부터 금융공학 알고리즘을 활용한 2030년 장기 예측까지 연결하여 경제/금융 탐구 보고서 및 대시보드로 활용하기 최적화되어 있습니다.

---

## 🚀 주요 기능 및 페이지 구성

사이드바 내비게이션을 통해 총 5개의 독립된 분석 및 탐구 단계를 제공합니다.

1. **🏠 홈 (환율 변화 개요)**
   * **최근 변화 지표:** 주요 통화별 최신 환율 수치 모니터링
   * **통계적 요약:** 관측치, 평균값, 변동성(표준편차), 최솟값, 최댓값을 통해 환율의 전반적인 수준 파악
   * **연동성 검증:** 피어슨 상관계수($Pearson\ Correlation$)를 활용한 통화 간 가치 변화의 동조화 경향성 분석
   * **변화율 리스크 비교:** 최근 150거래일간의 일일 변화율(수익률) 시계열 비교

2. **🔮 2030년 장기 환율 예측 (핵심 탐구)**
   * **금융공학 모델 적용:** 장기 예측 시 발생하는 통계적 왜곡을 방지하고 현실성을 높이기 위해 **기하 브라운 운동(Geometric Brownian Motion, GBM)** 모델 도입
   * **몬테카를로 시뮬레이션:** 과거 환율 변화의 무작위성과 고유 추세(Drift 및 Volatility)를 추출하여 미래 시나리오를 반복 연산
   * **확률적 변화 범위 제시:** 평균 추세 예측선과 함께 미래로 갈수록 누적되는 불확실성을 반영한 **상한/하한 변화 경계선(폭등·폭락 시나리오)** 시각화

3. **🇺🇸 미국 (원/달러)**
   * 실제 환율의 추세 변화와 20일·60일 이동평균선(MA) 분석 및 일일 환율 변화 폭 분포 히스토그램($Histogram$) 제공

4. **🇯🇵 일본 (원/100엔)**
   * 엔화 환율의 이평선 추세 변화 및 가독성을 높인 독립형 변화 빈도 분포 차트 제공

5. **🇨🇳 중국 (원/위안)**
   * 위안화 환율의 역사적 변화 추이 및 일일 변화율 분포 데이터 제공

## 🛠️ 핵심 해결 기술 (Tech Stack)

* **Streamlit Single-File Architecture:** 별도의 폴더 분리 없이 단일 소스 코드 내에서 `st.sidebar.radio` 제어문을 활용한 깔끔한 멀티 페이지 구현
* **기하 브라운 운동 모델 ($GBM$):** 외부 머신러닝 라이브러리 의존성 없이 `numpy` 수학 연산만을 활용해 월스트리트 표준의 장기 자산 가격 시뮬레이션 구현
* **경로 최적화 및 자동 로드:** 웹앱 구동 시 서버 내에 있는 한글 CSV 데이터를 파이썬이 자동으로 탐색하여 사용자의 수동 업로드 허들 제거
* **지능형 인코딩 예외 처리:** 한국은행 ECOS 데이터 등 한글 CSV 파일에서 자주 발생하는 인코딩 문제(`UnicodeDecodeError`)를 해결하기 위한 다중 디코딩 로직 기본 탑재
